COMO COMEÇAR A JOGAR? Um guia detalhado de 5 etapas para um início bem-sucedido.
PASSO #1
REGISTRO DE CONTA
Para acessar o jogo, você precisa registrar uma conta no Painel do Jogador.
Criar Conta
PASSO #2
DOWNLOAD DO CLINTE
Baixe o cliente do jogo da maneira conveniente para você.
TORRENT Google Drive 1 Google Drive 2 LINK Direto
Requisitos de Sistema
Mínimo:
Sistema operacional: Windows 7;
Processador: 1 GHz;
RAM: 4 GB;
Placa de Vídeo: 128 Mb;
Disco Rígido: 25 GB;
Internet: 64 Kbps.
Recomendado:
Sistema operacional: Windows 7;
Processador: 2.5 GHz;
RAM: 8 GB;
Placa de Vídeo: 256 Mb;
Disco Rígido: 30 GB;
Internet: 256 Kbps.
PASSO #3
Instalando o jogo
Extraia o cliente completo do jogo em um lugar conveniente para você como a área de trabalho.

Localize o cliente completo, e abra pela pasta Patcher com o arquivo patcher.exe

Após isso atualize o jogo normalmente e se necessário verifique o cliente. Se o cliente for baixado como um arquivo, use o WinRAR ou 7zip para descompactar os arquivos.
DOWNLOAD WINRAR DOWNLOAD 7ZIP
PASSO #4
Checar os drivers
Para que o cliente do jogo funcione corretamente, a placa de vídeo e os drivers DirectX mais recentes devem estar instalados em seu computador.
NVIDIA GEFORCE DIRECTX AMD RADEON
PASSO #5
PRONTO, VAMOS JOGAR
O cliente do jogo é iniciado através de um atalho na área de trabalho ou do arquivo Patcher.exe, se necessário, o cliente será atualizado automaticamente para o atual, desejamos boa sorte na sua nova jornada.
Caso tenha dúvidas entre em contato via ticket com a equipe do servidor no discord.
